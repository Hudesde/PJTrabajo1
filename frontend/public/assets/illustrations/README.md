# Ilustraciones

Coloca aquí las imágenes (PNG/SVG/WebP) que usa la interfaz. Mientras no existan,
la app muestra un placeholder elegante en su lugar (no se rompe nada).

Archivos esperados (puedes generarlos con IA y soltarlos con estos nombres):

- login-workspace.png    -> Panel izquierdo del login (productividad / oficina moderna).
- empty-tasks.png        -> Estado vacío del dashboard (sin tareas).
- tasks-dashboard.png    -> (Opcional) ilustración de cabecera del dashboard.

Recomendado: imágenes horizontales, fondo claro/transparente, estilo flat,
acordes a la paleta institucional (vino #9D2449 / dorado #B38E5D).
